Urządzenie Szerokość Główne Zmiany
Desktop > 1024px Pełny widok, sidebar zawsze widoczny
Tablet 768px - 1024px Zmniejszony sidebar, layout 1-kolumnowy
Mobile < 768px Hamburger menu, drawer navigation
Small Mobile < 480px Kompaktowe elementy, mniejsze czcionki
Extra Small < 360px Minimalne rozmiary, ultra-kompakt

💡 Wskazówki dla Użytkowników

  • Otwieranie menu: Kliknij ikonę 🍔 w lewym górnym rogu lub przeciągnij palcem od lewej krawędzi ekranu
  • Zamykanie menu: Kliknij poza menu, przeciągnij w lewo lub naciśnij ESC
  • Wyszukiwanie: Pole wyszukiwania dostosowuje się do szerokości ekranu
  • Nawigacja: Breadcrumbs można przewijać poziomo na małych ekranach
  • Karty: Wszystkie karty mają wizualny feedback przy dotknięciu

🔧 Optymalizacje Techniczne

Performance

  • Hardware-accelerated animations
  • Passive event listeners
  • Reduced motion support
  • Optimized scrolling

Accessibility

  • ARIA labels dla screen readers
  • Keyboard navigation (ESC key)
  • 48px minimum touch targets
  • High contrast support

Kompatybilność

  • iOS Safari 14+
  • Chrome Mobile 90+
  • Firefox Mobile 88+
  • Samsung Internet 14+

📱 Safe Area Support

Aplikacja respektuje safe areas dla urządzeń z notch (wcięcie):

  • viewport-fit=cover dla pełnoekranowego doświadczenia
  • env(safe-area-inset-*) dla padding adjustments
  • Optymalizacja dla iPhone X i nowszych

📖 Dalsze Kroki

Aby w pełni przetestować wersję mobilną:

  1. Otwórz aplikację na urządzeniu mobilnym
  2. Wypróbuj gestykulację dotykową (swipe)
  3. Przetestuj w różnych orientacjach (portrait/landscape)
  4. Sprawdź działanie na różnych rozmiarach ekranów
  5. Oceń responsywność i płynność animacji
